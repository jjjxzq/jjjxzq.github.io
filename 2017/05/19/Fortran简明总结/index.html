  • 自定义函数需要声明

common a,b;

声明成common的变量不能使用DATA
取用全局变量时根据其位置关系而不是变量名称来对应

  • 传递数组元素是本质上实在传递数组元素的内存地址
  • 一般在传递数组是最好也顺便输入它原来声明的大小

传递函数

被传递的函数要在主程序与子程序中被同时声明（就像变量一样）
传递函数的思想和传递数值的完全相同

特殊参数

####接口interface
用来说明所调用函数的参数类型及返回值类型等的使用接口

  • 函数返回值为数组
  • 指定参数位置传递参数
  • 调用的函数参数数目不固定
  • 输入指针参数
  • 函数的返回值为指针

返回值为数组

不定个数参数传递

递归

最开头加上recursive才可以进行递归调用
result可以用来在code中使用另外一个名字来设置函数的传回值

内部函数
contains；

pure函数（配合并行计算）

elemental函数（配合并行计算）

####module
用来封装程序模块

*在module声明中指定要save的变量，功能上也等同与全局变量

  • 通常会将功能上相关的函数放在同一个module中，函数可以直接使用同一个module里所声明的变量

少用的功能

  • entry
  • return

使用多个文件
include 'ex2015'； ！插入ex2015这个文件（必须在同一目录下）
在有了module后，就不太需要include了

指针

读写指针变量步骤：
1.读取指针中所保存的内存位置
2.到这个内存位置读取数据
来源：

  • 记录其它非指针变量的内存位置
  • 程序执行中动态配置一块内存

注意声明时两者都需要变量类型

allocate(p); !配置一块内存空间给指针p

指针变量本质上记录内存空间，因此在64位计算机上使用8bytes空间

####指针数组

  • 把指针指向其它数组
  • 配置内存空间

integer, pointer :: a(:,:)
指针数组在使用时只需要说明维数，不需要说明大小，类似可变数组
allocate配置的内存，记得使用deallocate释放回去

注意：声明成allocatable的数组有其生存周期，只存在于声明它的函数当中，函数结束后会自动deallocate
如果声明成指针，在函数结束后不会自动释放内存

指针与函数

  • 要把指针传递给参数时，要声明这个函数的参数使用接口interface
  • 指针参数声明时不需要intent这个形容词
  • 函数返回值若为指针时需要定义函数interface

###指针的应用

  • 方便使用高维数组中的某一部分元素

面向对象

再论module：

  • module可以声明变量
  • module可以自定义类型
  • module中可以编写函数
  • 同一module函数参数共享

public和private区分成公开使用及私下使用

同时使用多个module时会遇到变量或者函数名称重复，use后面可以临时改名
use A, aa=>va !把module A中的变量va改名为aa来使用
use A，only：vc ！只用module A中的变量vc
use A, only : c=>vc ！只用vc，把vc改成c

再论interface（用来说明函数参数及返回类型）：
同名函数的重载（overload）

####open命令

unit = number 必须是一个正整数，可以使用变量避开1,2,5,6

1,5是默认输入位置，2,6是默认输出位置

file = ‘filename’ 指定文件名

form = formarred/unformatted 文本文件和二进制文件（默认文本）

status = new/old/replace/scratch/unknown

access = sequential/direct 默认为sequential

recl = length 一次读取多大数据，字符（文本）或1/4bytes（二进制）
